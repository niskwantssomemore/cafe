# Лендинг кафе "Кофе & Книги"

Учебный проект по созданию адаптивного лендинга для местного кафе.

## Описание проекта

Лендинг для кафе "Кофе & Книги", включающий следующие разделы:
- Главная с Hero-секцией
- О кафе
- Интерактивное меню с табами
- Особенности заведения
- Форма бронирования столика
- Контактная информация

## Технологии

- **HTML5** с семантической разметкой
- **CSS3** с Flexbox и Grid для верстки
- **JavaScript** для интерактивности
- **Адаптивный дизайн** (mobile-first)
- **GitHub Pages** для хостинга

## Функциональность

1. **Адаптивный дизайн** - корректное отображение на всех устройствах
2. **Бургер-меню** для мобильной версии
3. **Табы** для переключения категорий меню
4. **Модальное окно** с подтверждением бронирования
5. **Слайдер фоновых изображений** на главном экране
6. **Валидация формы** бронирования
